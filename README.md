# TillMate — UK Coin Weight Counter

A lightweight, mobile-friendly web app to estimate UK coin counts and cash totals using coin weight in grams. Useful for retail / post office cash-up.

## Features
- Enter per-coin weight totals (g) and optional tare (container weight)
- Calculates coin count per denomination and totals (£)
- Grand total coins + amount
- Rounding modes: Round / Floor / Ceil
- Editable coin weight settings + persistent save (localStorage)
- Responsive UI (Bootstrap)

## Tech Stack
HTML • CSS • Bootstrap 5 • JavaScript • jQuery
